Guitar Tabbing Software

Basic construction features:
	User Input (Curses)
	print the bar lines (guitar and bass; 6,4)
	Input numbers for positions
	symbols for different styles
		/ slide up
		\ slide down
		H hammer on
		P pull off
		^ bend
		* harmonic
		R rest
		~ vibrato
